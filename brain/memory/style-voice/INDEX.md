## The Three-Mode System

```
Context Assessment
    ↓
Mode Selection (Casual/Professional/Formal)
    ↓
Apply Tone + Vocabulary + Structure
    ↓
Create Communication
```

### Quick Mode Selector

**Ask yourself:**
1. **Who's the audience?**
   - Just me → Casual
   - Client/colleague → Professional
   - Executive/official → Formal

2. **What's at stake?**
   - Low → Casual
   - Medium → Professional
   - High → Formal

3. **What's the context?**
   - Internal → Casual
   - Standard business → Professional
   - Official/compliance → Formal

## Supporting Executive Function

### When You're Stuck (Blank Page Paralysis)

**Quick Protocol**:
1. Open structure.md
2. Find template matching your communication type
3. Copy the template
4. Fill in your content (even rough bullets)
5. Use casual mode first if easier
6. Upgrade to professional/formal later if needed

**Key Principle**: Something is better than nothing. Lower the bar, get started, refine later.

### When Context Switching Between Clients

**Quick Protocol**:
1. Check which mode this client relationship uses
2. Reference examples.md for similar past communication
3. Copy structure, update content
4. Maintain consistency within each client relationship

### When You Need AI Help

**Quick Protocol**:
1. Specify the mode: "Write this in professional mode"
2. Reference relevant files: "Use tone-guide.md and structure.md"
3. Provide context: "This is for [client/situation]"
4. Review and refine output

## Using This Directory with AI

### For Communication Creation
**Prompt**: "Using my style-voice/ directory, write a [professional/formal/casual] [email/proposal/summary] about [topic]."

The AI will:
- Select appropriate mode from tone-guide.md
- Apply mode-specific vocabulary
- Use relevant structure template
- Reference similar examples

### For Overcoming Blank Page
**Prompt**: "I need to write [communication type] but I'm stuck. Give me a structure and draft in [mode]."

The AI will:
- Pull template from structure.md
- Create rough draft in specified mode
- Provide something to work with
- Lower the cognitive load

### For Mode Selection Help
**Prompt**: "Should this be casual, professional, or formal? Context: [describe situation]"

The AI will:
- Assess context against mode criteria
- Recommend appropriate mode
- Explain reasoning
- Reference tone-guide.md

### For Voice Refinement
**Prompt**: "Review this draft against my style-voice/ directory. Make it match the [mode] voice better."

The AI will:
- Check tone alignment with mode
- Adjust vocabulary to match preferences
- Ensure structure is appropriate
- Refine for consistency

- [ ] Accomplishes communication goal
